PaketTemplateParams Type

Namespace: Fake.DotNet
Assembly: Fake.DotNet.Paket.dll
Parent Module: PaketTemplate
Base Type: obj
All Interfaces: IEquatable<PaketTemplateParams> , IStructuralEquatable , IComparable<PaketTemplateParams> , IComparable , IStructuralComparable

Contains the different parameters to create a paket.template file
