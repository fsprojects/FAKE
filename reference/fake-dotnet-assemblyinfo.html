AssemblyInfo Module

Namespace: Fake.DotNet
Assembly: Fake.DotNet.AssemblyInfoFile.dll

Contains tasks to generate AssemblyInfo files for C# and F#. There is also a tutorial about the AssemblyInfo tasks available.
